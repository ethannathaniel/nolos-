Información

son un conjunto de datos entre lazados mediante conectores lingüisticos para formar una idea o definición.

Base de Datos

Es un conjunto de datos pertenecientes a un mismo contexto y almacenados sistematicamente para su posterior uso

Servidor de base de Datos

son los que se utilizan para los gestores de bases de datos y que varios usuarios hagan varias operaciones sobre ellas situadas en lugares diferentes.
se pueden acceder a ellas mediante terminales o equipos con un programa llamado cliente.

Gestor de base de datos

son un conjunto de programas que manejan todo acceso a la base de datos, con el objetivo de servir de interfaz entre esta, el usuario y la aplicaciones utilizadas.
